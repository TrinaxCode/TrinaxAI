# Código de Conducta

[English](../../CODE_OF_CONDUCT.md)

TrinaxAI da la bienvenida a contribuidores que quieran mejorar la IA local-first.

## Comportamiento esperado

- Sé respetuoso y directo.
- Mantén las discusiones técnicas y constructivas.
- Asume buena intención, pero acepta correcciones rápidamente.
- Reporta problemas de seguridad de forma privada como se describe en `SECURITY.md`.

## Comportamiento inaceptable

- Acoso, amenazas, discurso de odio o ataques personales.
- Publicar información privada sin permiso.
- Spam, troleo o desviar intencionalmente issues y pull requests.
- Presionar a los mantenedores para ejecutar comandos inseguros o aceptar código no revisado.

## Aplicación

Los mantenedores pueden editar, ocultar o eliminar comentarios, issues, pull requests u otras contribuciones que violen este código. Las violaciones repetidas o graves pueden resultar en la expulsión de los espacios del proyecto.

## Reporte

Para situaciones que puedan discutirse públicamente, abre un issue en GitHub. Para violaciones de conducta privadas o informes sensibles, envía un correo a [trinaxcode@gmail.com](mailto:trinaxcode@gmail.com) directamente en lugar de usar un issue público. Para vulnerabilidades de seguridad, sigue `SECURITY.md`.
