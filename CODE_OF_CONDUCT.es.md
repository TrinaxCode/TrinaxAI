# Código de Conducta

TrinaxAI da la bienvenida a colaboradores que quieran mejorar la IA local.

## Comportamiento esperado

- Sé respetuoso y directo.
- Mantén las discusiones en un tono técnico y constructivo.
- Asume buenas intenciones, pero acepta correcciones con rapidez.
- Reporta los problemas de seguridad de forma privada como se describe en `SECURITY.md`.

## Comportamiento inaceptable

- Acoso, amenazas, discurso de odio o ataques personales.
- Publicar información privada de otras personas sin su permiso.
- Spam, trolling o descarrilar intencionadamente issues y pull requests.
- Presionar a los mantenedores para que ejecuten comandos inseguros o acepten código sin revisar.

## Aplicación

Los mantenedores pueden editar, ocultar o eliminar comentarios, issues, pull requests u otras contribuciones que infrinjan este código. Las violaciones reiteradas o graves pueden derivar en una prohibición de acceso a los espacios del proyecto.

## Reportar una infracción

Para asuntos que sea seguro tratar públicamente, abre un GitHub issue. Para violaciones de conducta privadas o reportes sensibles, envía un correo directamente a [security@trinaxcode.com](mailto:security@trinaxcode.com) en lugar de usar un issue público. Para vulnerabilidades de seguridad, sigue las instrucciones de `SECURITY.md`.
