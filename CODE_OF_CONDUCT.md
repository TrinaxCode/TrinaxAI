# Code of Conduct

[Versión en español](docs/es/CODE_OF_CONDUCT.md)

TrinaxAI welcomes contributors who want to improve local-first AI.

## Expected Behavior

- Be respectful and direct.
- Keep discussions technical and constructive.
- Assume good intent, but accept correction quickly.
- Report security issues privately as described in `SECURITY.md`.

## Unacceptable Behavior

- Harassment, threats, hate speech, or personal attacks.
- Publishing private information without permission.
- Spam, trolling, or intentionally derailing issues and pull requests.
- Pressuring maintainers to run unsafe commands or accept unreviewed code.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or other contributions that violate this code. Repeated or severe violations may lead to a ban from project spaces.

## Reporting

For concerns safe to discuss publicly, open a GitHub issue. For private conduct violations or sensitive reports, email [trinaxcode@gmail.com](mailto:trinaxcode@gmail.com) directly instead of using a public issue. For security vulnerabilities, follow `SECURITY.md`.
