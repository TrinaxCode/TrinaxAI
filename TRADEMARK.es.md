# Política de Marca de TrinaxAI

TrinaxAI es el nombre de este proyecto y producto de código abierto. El nombre TrinaxAI es una marca registrada de TrinaxCode (<https://github.com/TrinaxCode>).

Puedes usar el nombre TrinaxAI para:

- Hacer referencia al proyecto oficial.
- Enlazar al repositorio oficial.
- Describir compatibilidad con TrinaxAI.
- Publicar forks que indiquen claramente que son forks o versiones modificadas.

No puedes usar el nombre ni los logotipos de TrinaxAI para:

- Insinuar que un fork no oficial, servicio, paquete o compilación es el proyecto oficial de TrinaxAI.
- Confundir a los usuarios sobre quién mantiene o asegura un lanzamiento.
- Vender servicios hospedados o binarios usando el nombre TrinaxAI sin una atribución clara y permiso expreso.

Los forks deben usar nombres como "Fork of TrinaxAI" o "Powered by TrinaxAI" y deben enlazar claramente de vuelta al repositorio oficial.

Esta política trata sobre la identidad del proyecto. No modifica la licencia del software descrita en `LICENSE`.
